# basic-dockerfile  

Project: [https://roadmap.sh/projects/basic-dockerfile](https://roadmap.sh/projects/basic-dockerfile)

## ⚙️ Project Setup
```bash
git clone https://github.com/Bohdan14228/basic-dockerfile
cd basic-dockerfile
docker build -t hello .
docker run --name hello hello (default)
docker run --name hello hello "YOUR_NAME"
```
